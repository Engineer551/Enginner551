##### Key UX Improvements:
- **Mobile-Responsive First Layout:** Built natively for students accessing the portal via smartphones.
- **Role-Based Tab Navigation:** Clean segment selection to quickly toggle between Student, Personnel, and Alumni logins.
- **Enhanced Readability:** Implemented clear typographic contrast and proper form spacing to reduce cognitive overload.
